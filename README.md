# Three-Tier Application Infrastructure (Terraform)

This project provisions a three-tier architecture on AWS using Terraform.

## Project Details
- Project: three-tier-app
- Environment: dev
- Region: ap-south-1

## Managed Services
- VPC (public & private subnets)
- Application Load Balancer
- EC2 (frontend & backend)
- RDS (database)
- AWS Secrets Manager
- S3 (Terraform state)
